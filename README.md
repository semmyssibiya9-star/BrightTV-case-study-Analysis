
Bright TV Case Study: Viewership and Performance Analysis

Overview:

This repository contains a detailed analysis of Bright TV’s viewership trends, engagement metrics, and content performance. The purpose of this case study is to identify actionable insights for enhancing audience retention, optimizing programming strategy, and maximizing overall platform engagement.

Objectives:

Examine viewership patterns across different time slots and content categories.
Analyse audience demographics and engagement behaviours.
Assess the effectiveness of marketing campaigns on viewer acquisition and retention.
Provide recommendations for data-driven decision-making to improve Bright TV’s performance.
Dataset
The analysis utilizes data collected from Bright TV’s viewing logs, including:
Date and time of viewership sessions
User demographics (age, location, subscription type)
Program metadata (duration)
Engagement metrics (playbacks, average watch time, likes, shares)
Marketing campaign records (email blasts, promotions, notifications)
Methodology
Data Cleaning and Preprocessing: Handle missing values, normalize time stamps, and categorize content.
Exploratory Data Analysis (EDA): Identify trends, peaks, and anomalies in viewership using visualizations and summary statistics.
Segmentation Analysis: Segment audience based on demographics and engagement to uncover insights tailored for each viewer group.
Correlation and Impact Analysis: Evaluate factors driving engagement and retention, including time slots, program type, and marketing efforts.
Predictive Modelling (Optional): Build models to forecast viewership trends or churn probabilities.

Tools and Technologies:

SQL (Databricks, Biq query)
Excel or CSV format for raw datasets
Optional: Tableau/Power BI/ Looker Studio for advanced visual dashboards. Miro for project planning, creating a flowchart diagram and creating a timeline project planning. 

Key Findings (Summary):

Highlight the programs with highest and lowest engagement rates.
Identify optimal broadcast times for target demographics.
Reveal the impact of marketing campaigns on viewer retention.
Provide data-driven recommendations to improve content strategy, increase watch time, and enhance overall audience satisfaction.

Recommendations:

Optimize Content Scheduling and Programming. 
Enhance Recommendation and Personalization.
 Expand Market Reach.
Key Performance Indicators (KPIs) to Track.
Improve User Engagement and Retention.
